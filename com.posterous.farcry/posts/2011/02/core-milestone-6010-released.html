Core Milestone 6.0.10 Released

February 22 2011, 11:21 PM  by Geoff Bowers

The complete list of changes for 6.0.x maintenance branch is available online.

Download the ZIP for for the 6.0.10 core milestone here: http://org.farcrycore.s3.amazonaws.com/core/m6010.zip

Or alternatively you can update from Subversion (SVN) at this address:
svn checkout https://farcry.jira.com/svn/FC/tags/milestone-6-0-10

Bug

  • [FC-2443] - Regenerate image doesn't allow immediate preview like it did previously
  • [FC-2444] - Data currpution when attaching newly created items to a single select formtool (UUID)
  • [FC-2446] - illegal filename can cause JSON format error for image formtool
  • [FC-2447] - Images with quotes in the names (e.g. china's great wall.jpg) aren't handled well
  • [FC-2448] - References to 404.cfm in the display tag won't work for standalone installations

Improvement

  • [FC-1943] - Object Admin Filter - Apply filter when pressing ENTER

Enjoy!
