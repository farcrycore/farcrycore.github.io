Core Milestone 6.0.9 Released

February 22 2011, 10:44 PM  by Geoff Bowers

The complete list of changes for 6.0.x maintenance branch is available online.

Download the ZIP for for the 6.0.9 core milestone here: http://org.farcrycore.s3.amazonaws.com/core/m609.zip

Or alternatively you can update from Subversion (SVN) at this address:
svn checkout https://farcry.jira.com/svn/FC/tags/milestone-6-0-9
