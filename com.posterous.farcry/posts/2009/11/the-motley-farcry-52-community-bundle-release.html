The Motley FarCry 5.2 Community Bundle Released

November 13 2009, 9:43 PM  by Geoff Bowers
Although 5.2 has been available for some time on Subversion, we wanted to iron out a host of edge cases surrounding he changes in "friendly urls" before unleashing it all on the wider community. FarCry 5.2, the "Motley" release, is an awesome array of behind the scenes improvements, refactoring and bug fixes.

Notable changes include:

* Friendly URL Engine; constructed URLs, FU's plus URL parameters
* Type level "On Change" Events; big extension on beforeSave() & afterSave()
* Media Management; secure & unsecured files
* Javascript & CSS Loading; combine functions (aggregate, minify, cacheing)
* Plus a motley collection of other enhancements

You can get it here: http://farcrycore.org/download
