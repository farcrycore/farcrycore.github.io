FarCry Builder v1.0: Content Type Magic

October 24 2012, 11:48 PM  by Geoff Bowers

Daemonite Justin has been beavering away on a pet project for many moons. It's pretty special.

FarCry Builder is a Chrome extension written in Javascript for building content types. Sounds simple enough, but once you open up this little toolkit you get a feel for just how cool this mini-IDE really is.

Modify existing content types or build them from scratch. Put content types together visually with a drag'n'drop from the Formtools Palette. Marvel at the beautifully formatted code written for you. Or if you prefer, code away and see the content type visualised in real time. FarCry Builder supports nearly all the standard formtools, fieldsets and wizard steps, plus a bunch of common formtool combinations like image cropping.

Download and install by following the instructions on the FarCry Builder wiki page:

  • https://farcry.jira.com/wiki/display/BLD 

 Enjoy!
