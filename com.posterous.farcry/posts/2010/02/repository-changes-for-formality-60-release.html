Repository Changes for Formality (6.0) Release

February 21 2010, 4:34 AM  by Geoff Bowers

Please be aware that there have been a variety of repository changes to accompany the 6.0 release.

./core/branches/p600
----------------------------------
p600 is the maintenance branch for the 6.0.x release. Please switch your workspaces to this location rather than pointing at trunk. All maintenance patches for 6.0 should be committed to this branch (not to trunk).

https://modius.svn.cvsdude.com/farcry/core/branches/p600

./core/tags/milestone-6-0-0
----------------------------------
milestone-6-0-0 is the tagged release for 6.0.0. Please shift production clients to this release as needed.

https://modius.svn.cvsdude.com/farcry/core/tags/milestone-6-0-0

./core/trunk
----------------------------------
trunk is now the official mainline for 6.1.0 (codename: Mayonnaise) development only. Please do *not* commit maintenance changes for 6.0.x to the trunk, use the maintenance branch instead. Patches to p600 will be periodically merged into trunk.

https://modius.svn.cvsdude.com/farcry/core/trunk

./distro/community/tags/release-6-0-0
----------------------------------
The community distribution workspace has been tagged. Point people here who want to download the release via Subversion. This is what I'm using to build the community distro zip file.

https://modius.svn.cvsdude.com/farcry/distro/community/tags/release-6-0-0

./distro/community/trunk
----------------------------------
The community distribution workspace nightly, ie. the latest of everything, is available for those wanting to live dangerously.

https://modius.svn.cvsdude.com/farcry/distro/community/trunk
