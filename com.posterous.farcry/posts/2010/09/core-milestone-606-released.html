Core Milestone 6.0.6 Released

September 24 2010, 1:03 AM  by Geoff Bowers

The complete list of changes for 6.0.6 maintenance release is available here under "Filters > All":
https://farcry.jira.com/browse/FC/fixforversion/10361

Download the ZIP for for the 6.0.6 core milestone here:
http://org.farcrycore.s3.amazonaws.com/core/m606.zip

Or alternatively you can update from Subversion (SVN) at this address:
svn checkout https://farcry.jira.com/svn/FC/tags/milestone-6-0-6

** Bug Fixes

* [FC-1761] - Code error in core/webskin/farUser/forgotUserID.cfm
* [FC-2292] - Scaffolding doesn't work
* [FC-2323] - Extended Array Table no longer go into refObjects.
* [FC-2324] - State formtool throws an error when displaying
* [FC-2326] - longchar.cfc - ftLimit needs default of zero not empty tring
* [FC-2328] - If sqlWhere is empty in formtool then causes error
* [FC-2334] - ft:button attribute bSpamProtect does not always work
* [FC-2335] - Change password link on webtop overview screen is broken
* [FC-2338] - Form select option tags not standardized
* [FC-2342] - replace image option prepends image filename with date_time instead of archiving old image
* [FC-2347] - download.cfm links are being processed by farFU

** Improvement

* [FC-1469] - File and Image formtool types need a accept filetype property (ftFileTypes)
* [FC-2325] - Button form tool jquery no referencing '$j' namespace
* [FC-2329] - Allow plugins to be loaded from different directories
* [FC-2331] - cacheByVars does not handle cfparam'd variables
* [FC-2332] - ftlibrarydatasqlorderby defaulting to label in join.cfc - better workflow is to show latest modified first
* [FC-2333] - getContentObjects should ignore empty category filters
* [FC-2339] - Enhance library picker JS to pass URL parameters along
* [FC-2348] - Allow "packaging" of registered JS/CSS into a single file
* [FC-2350] - If there is only one rule in lRules attribute for the container tag then you should be directed straight to the rule instead of selecting it from the list

** New Feature

* [FC-103] - Autogenerate Google sitemap protocol
* [FC-2330] - Simple request profiling and logging
* [FC-2349] - Add hostname attribute to loading and registering css and js libraries

Enjoy!
