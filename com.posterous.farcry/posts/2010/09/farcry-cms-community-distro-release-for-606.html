FarCry CMS Community Distro Release for 6.0.6

September 24 2010, 1:44 AM  by Geoff Bowers
Along with the recent Core 6.0.6 framework release, we've also updated the popular FarCry CMS Community distribution with all the latest CMS plugin and sample project.

You can get the latest distro download here:
http://org.farcrycore.s3.amazonaws.com/release/community-6-0-6.zip

Or direct from the repository workspace here:
svn checkout https://farcry.jira.com/svn/REL/tags/community-6-0-6

Enjoy!
