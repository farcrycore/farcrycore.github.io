FarCry Formtools Moves to JQuery UI

January 27 2010, 8:40 PM  by Geoff Bowers
One of the major changes in FarCry 6.0 (codenamed: Formailty) is the move to the exclusive use of JQuery and JQuery UI in the Webtop.  JQuery UI is being used as the standard for all the UI elements - well at least that is the direction we're headed.  Some parts of the Webtop chrome will retain the original blue/grey "Glamour" release design.

The default UI theme we're using in the beta is a Silver/Grey called "Smoothness" -- it's unobtrusive and works well.

Have you got a better colour scheme in mind? Try your hand at theme building and let us know: http://jqueryui.com/themeroller/
