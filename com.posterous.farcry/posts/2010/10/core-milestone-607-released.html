Core Milestone 6.0.7 Released

October 11 2010, 5:34 AM  by Geoff Bowers
The complete list of changes for 6.0.7 maintenance release is available here under "Filters > All": 
https://farcry.jira.com/browse/FC/fixforversion/10362

Download the ZIP for for the 6.0.6 core milestone here: 
http://org.farcrycore.s3.amazonaws.com/core/m607.zip

Or alternatively you can update from Subversion (SVN) at this address: 
svn checkout https://farcry.jira.com/svn/FC/tags/milestone-6-0-7

** Bug
    * [FC-200] - dmCSS edit handler breaks if underlying CSS file is missing from file system
    * [FC-530] - dmCSS: Can't upload CSS stylesheet
    * [FC-959] - Delete image from sitetree don't work (dmImage.cfc - delete)
    * [FC-992] - Resorting Library Items Sa ves With out Confirmation
    * [FC-1437] - container: cancelling container editing should not reload the container
    * [FC-2175] - Update app not working from tray
    * [FC-2235] - Can't 'detach' objects from rules picker
    * [FC-2296] - Case sensitivity issue with farFU
    * [FC-2355] - Overview dashboard shows only draft items the current user owns
    * [FC-2360] - Redirected back to home page after logging into webtop in subdirectory install
    * [FC-2362] - Incorrect path for dmImage thumbnail on webtopOverviewSummary webskin
    * [FC-2363] - all jquery references should used jquery or $j
    * [FC-2371] - Friendly URL is not parsed correctly if it begins with two slashes (//)
    * [FC-2372] - farLogin form does not redirect user pr operly after switching projects or UD

** Improvement
    * [FC-1796] - Allow the container description to be different to the actual label.
    * [FC-1832] - skin:view default to typename of dmHTML
    * [FC-2273] - skin:genericNav new attribute to allow tag to work with categories (fix incl.)
    * [FC-2319] - farWebfeed.getFeedObjects() need to filter on status field
    * [FC-2354] - There should be a simple way to handle skin:bubble messages inline
    * [FC-2359] - Need to allow custom configuration for richtext on a per-field basis
    * [FC-2364] - add title attribute to ft:button
    * [FC-2366] - New layout for objectadmin action column

** New Feature
    * [FC-2356] - Allow for a customized webto p/overvi ew/home.cfm
    * [FC-2365] - grid:div will now render any attribute passed in as an html attribute

Enjoy!
