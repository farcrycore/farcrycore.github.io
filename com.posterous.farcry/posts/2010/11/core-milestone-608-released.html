Core Milestone 6.0.8 Released

November 30 2010, 11:02 PM  by Geoff Bowers

The complete list of changes for 6.0.8 maintenance release is available here under "Filters > All":
https://farcry.jira.com/browse/FC/fixforversion/10370

Download the ZIP for for the 6.0.8 core milestone here:
http://org.farcrycore.s3.amazonaws.com/core/m608.zip

Or alternatively you can update from Subversion (SVN) at this address:
svn checkout https://farcry.jira.com/svn/FC/tags/milestone-6-0-8

** Bug
   * [FC-2191] - Login to webtop redirects to front end
   * [FC-2376] - Error when filtering an ntext field with objectadmin
   * [FC-2385] - image formtool stSupporting has no "DELETE" key< br>  ;  * [FC-2389] - Hardcoded reference to /webtop instead of application.url.webtop
   * [FC-2392] - Element BSUCCESS is undefined in STGENERATEDIMAGE

** Improvement
   * [FC-295] - Image upload file size restriction
   * [FC-2379] - Ability to easily add tabs to object overview page
   * [FC-2383] - Object admin filters should also include matching editable drafts in results
   * [FC-2386] - Upgrade TinyMCE to latest version (3.3.9.2) using jQuery plugin method
   * [FC-2387] - Update jQuery to 1.4.3
   * [FC-2388] - The "None" resize option in the image formtool shouldn't resave the image 
   * [FC-2390] - Implement new, slicker FarCry Tray

** New Feature
   * [FC-2384] - Ability to add watermark to image formtool

** Task
   * [FC-2393] - Railo 3.2 - n ew function arrayReverse

Enjoy!
